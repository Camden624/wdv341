Git Terms

  • Version Control Software: This is software that allows you to store your code in a safe space (online or locally), and view the changes that you have made to the code. Other people can collaborate on the same code, and the software will keep track of who made what changes.
  • Add: This command is used to add files to the staging area. The staging area is where you put files that you want to commit to the repository.
  • Commit: This command is used to save changes to the repository. When you commit changes, you are saving a snapshot of the current state of the code.
  • Push: This command is used to upload changes to the online repository. When you push changes, you are sending the changes you made on your local machine to the online repository.
  • Pull: This command grabs the latest changes from the online repository and merges them with your local repository.
  • Clone: This command is used to copy a repository from an online location to your local machine.
